API Reference

Propriedade Tipo Padrão Descrição
[options] SelectOption[] [] Lista de opções disponíveis para seleção.
[label] string '' Texto do rótulo exibido acima do componente.
[placeholder] string 'Selecione...' Texto exibido quando nenhum valor está selecionado.
[disabled] boolean false Desabilita a interação com o componente.
[error] boolean false Aplica estilos visuais de erro (borda vermelha).
(change) EventEmitter<any> - Evento emitido quando o valor selecionado muda.
